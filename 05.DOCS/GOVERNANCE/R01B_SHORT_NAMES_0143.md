# R01B short names policy 0143

- Maximum recommended base name for CSV/log outputs: `40` characters before extension.
- Preferred pattern: `R01B_<TOPIC>_<WBS>.csv`.
- Examples:
  - `R01B_CACHE_0143.csv`
  - `R01B_GATE_0143.csv`
  - `R01B_FIX_0143.csv`
  - `R01B_TREE_0143.csv`
- Legacy long names are not renamed in place in this WBS.
- Map long names in inventory first, then archive or alias in a future hygiene WBS.
