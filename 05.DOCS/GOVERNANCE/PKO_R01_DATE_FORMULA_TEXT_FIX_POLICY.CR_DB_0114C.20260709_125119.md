# PKO.R01 DATE formula text fix policy - CR-DB-0114C

- MB_ID: CR-DB-0114C-DATE-FORMULA-TEXT-TRACE-AND-FIX
- Status: ACTIVE

If a PKO output displays DATE formulas as visible text, the cell must be converted to a real Excel formula and recorded in DB fix registry.

This task does not infer new date logic from labels. It fixes explicit DATE formula text detected in outputs.
