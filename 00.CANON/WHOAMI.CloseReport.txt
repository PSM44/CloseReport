OWNER.............: Pablo (SN Holding — Kumquat + KMT)
ROOT..............: C:\01. GitHub\CloseReport
REPO..............: github.com/[usuario]/CloseReport

PROPOSITO.........:
Reemplazar el proceso manual de armado del Comité de Finanzas
mensual de SN Holding. Hoy ese proceso requiere copiar datos entre
~10 archivos Excel distintos, con fórmulas rotas, categorías
inconsistentes y sin fuente única de verdad.

CloseReport establece una base de datos central que alimenta
todos los reportes del comité automáticamente al refrescar.

COMO_FUNCIONA.....:
  Fuente (Excel raw)
    → ETL Python (clean + normalize + export)
      → CSV staging
        → Excel Power Query (CloseReport.xlsm)
          → Slides del Comité de Finanzas

TECNOLOGIAS.......:
  Python 3.x + pandas + openpyxl
  PowerShell (orquestador)
  Excel Power Query (inglés)
  CSV (fase 1-2) → SQLite (fase 3-5) → PostgreSQL (fase 6+)
  VS Code + Git
